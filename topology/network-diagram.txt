DHCP FLOW:
==========
1. Client in VLAN X requests IP
2. Request hits VLAN interface on Cairo switch
3. DHCP relay (ip helper-address 10.2.2.2) forwards to Luxor
4. Luxor DHCP server responds with:
   - IP from appropriate range (.11-.100)
   - Gateway = VLAN interface (.1)
   - DNS = 8.8.8.8, 8.8.4.4
   - Domain = netinfra.local
